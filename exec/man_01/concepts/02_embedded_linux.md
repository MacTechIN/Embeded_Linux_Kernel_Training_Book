# 개념 02: 임베디드 리눅스

## 한 줄 정의

> **임베디드 리눅스** = 임베디드 시스템의 자원 제약과 목적에 맞춰 **잘게 다듬어 올린 리눅스 운영체제**.

## "다듬는다"는 의미: Tailoring

양복을 사람 몸에 맞춰 줄이고 늘리듯이, 리눅스를 하드웨어와 용도에 맞춰:
- 불필요한 드라이버 제거
- 불필요한 프로그램 제거
- 부팅 속도 최적화
- 메모리 사용 최소화

## 일반 리눅스 vs 임베디드 리눅스

### 이미지 크기 비교

| 항목 | Ubuntu Desktop | 임베디드 리눅스 |
|------|-----------------|-------------------|
| 설치 이미지 | ~4 GB | 1~8 MB |
| 설치 후 크기 | 8~15 GB | 8~64 MB |
| 비율 | 1배 | **1/100 ~ 1/200배** |

### 부팅 시간 비교

| 항목 | Ubuntu Desktop | 임베디드 리눅스 |
|------|-----------------|-------------------|
| 일반적 부팅 | 30~60초 | 1~5초 |
| 산업용 부팅 | - | **0.5초 이하** |

### 포함되는 프로그램 비교

#### Ubuntu Desktop (대표 프로그램)
- 데스크톱 환경 (GNOME, KDE, 등)
- 브라우저 (Firefox, Chrome)
- 오피스 (LibreOffice)
- 멀티미디어 (VLC, GIMP)
- 게임

#### 임베디드 리눅스 (최소 구성)
- 커널 (Linux Kernel)
- 최소 C 라이브러리 (libc 또는 musl)
- 필수 명령 유틸리티 (ls, cp, mount 등)
- 네트워크 스택
- 특정 응용 프로그램 (웹 서버, 센서 드라이버 등)

## 임베디드 리눅스가 사용되는 곳

| 분야 | 예시 |
|------|------|
| 모바일 | Android (실제로 Linux 커널 기반) |
| 네트워크 | 라우터, 스위치, 방화벽 |
| 가전 | TV, 냉장고, 에어컨, 세탁기 |
| 자동차 | 인포테인먼트, 주행 제어 |
| 산업 | PLC, 제조 장비 |
| IoT | 스마트 센서, 스마트 홈 |
| 통신 | 기지국, 교환기 |
