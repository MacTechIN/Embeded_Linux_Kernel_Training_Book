# 개념 01: 임베디드 시스템이란

## 한 줄 정의

> **임베디드 시스템** = 어떤 기기의 일부로 내장되어, 제한된 자원(CPU·메모리·전력) 안에서 정해진 기능을 수행하도록 설계된 컴퓨터 시스템.

## 핵심 키워드 3가지

### 1. 내장(Embedded)
- 사용자가 "컴퓨터"로 인지하지 않는다
- 세탁기 내부의 컴퓨터 → 세탁기라고 부른다
- 자판기 내부의 컴퓨터 → 자판기라고 부른다

### 2. 특정 목적(Special-Purpose)
- 한 가지 일을 잘 해내도록 설계
- 세탁기는 세탁만 한다
- 도어록은 출입 제어만 한다

### 3. 자원 제약(Resource-Constrained)
- 일반 PC보다 훨씬 적은 메모리
- 적은 저장 공간
- 적은 전력 사용

## 일상 속 임베디드 시스템 예시

| 기기 | 내장 컴퓨터의 역할 |
|------|------------------|
| 디지털 시계 | 시간 표시, 알람 관리 |
| 버스 카드 단말기 | 카드 읽기, 요금 계산 |
| 도어록 | 인증, 잠금/해제 제어 |
| 자판기 | 상품 선택, 결제 처리 |
| ATM | 카드 읽기, 거래 처리 |
| 세탁기 | 세제 분사, 회전 제어 |
| IPTV 셋톱박스 | 채널 선택, 영상 디코딩 |
| 자동차 | 엔진 제어, 안전 장치 |
| 스마트폰 | 통신, 앱 실행 |
