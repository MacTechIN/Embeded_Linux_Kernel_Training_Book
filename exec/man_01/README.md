- [ ] 일반 PC 리눅스와 임베디드 리눅스의 차이를 세 가지 이상 설명할 수 있다.
- [ ] 7개 핵심 용어(host, target, cross-compile 등)를 설명할 수 있다.

2. **references/** — 추가 학습에 유용한 링크와 외부 자료입니다.
3. **notes/** — 학습 중 자신의 이해도를 정리하는 공간입니다.
