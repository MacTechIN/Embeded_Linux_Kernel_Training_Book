## 공식 문서 및 튜토리얼

### 일반 임베디드 리눅스

- [Embedded Linux Wiki](https://elinux.org/)
  - 임베디드 리눅스 개발자들의 공동 위키
  - 다양한 보드 정보, 프로젝트 가이드

- [Linux Foundation - Embedded Linux](https://www.linuxfoundation.org/)
  - 공식 교육 과정
  - 인증 프로그램

### 실습용 보드 문서

- [Raspberry Pi 공식 문서](https://www.raspberrypi.com/documentation/)
  - 설치, 사용법, GPIO 제어 등

- [BeagleBone 공식 사이트](https://beagleboard.org/)
  - 산업용 임베디드 개발 보드

## 도구 및 프로젝트

### 크로스 컴파일 및 빌드

- [Buildroot](https://buildroot.org/)
  - 임베디드 리눅스 루트파일시스템 생성 도구
  - 설정 간단, 초보자 추천

- [Yocto Project](https://www.yoctoproject.org/)
  - 산업용 임베디드 개발 플랫폼
  - 복잡하지만 강력함

- [U-Boot](https://www.denx.de/wiki/U-Boot)
  - 가장 인기 있는 부트로더

### 에뮬레이터

- [QEMU](https://www.qemu.org/)
  - CPU 아키텍처 에뮬레이터
  - ARM, x86, MIPS 등 지원
